# Quản lý quán cafe KEYCODE MON :coffee:
## Giới thiệu đề tài
Website "Quản lý quán cafe KEYCODE MON" là trang web hỗ trợ quản lý cho quán cafe sân vườn với các chức năng như:
- Nhân viên
	+ Đăng nhập / Đăng xuất
	+ Đổi thông tin tài khoản
	+ Thêm món vào bàn
	+ Chuyển bàn
	+ Thanh toán
	+ Hỗ trợ giảm giá
	+ Sao lưu dữ liệu
	+ Phục hồi dữ liệu
- Quản trị
	+ Các chức năng của nhân viên
	+ Quản lý tài khoản
	+ Quản lý bàn
	+ Quản lý món
	+ Quản lý danh mục
	+ Thống kê doanh thu
## Công nghệ sử dụng
